# Server-Authoritative Underway

Admission requires a live world presence in the authored Black Market route region. The server issues a short-lived session token tied to the source land.

The server owns:

- daily offer rotation;
- reputation requirements;
- crystal costs;
- contraband rewards;
- discovered routes;
- Lost Territory reveals;
- Forbidden Dungeon Keys;
- Anonymous Contracts;
- inspection protection.

Purchases use canonical inventory revision and idempotency receipts. A Forbidden Dungeon Key is consumed atomically when the player starts a keyed authoritative Dungeon run, and its sealed chest is owned by that run. An Anonymous Contract is consumed and rewarded only while committing the final floor receipt.
