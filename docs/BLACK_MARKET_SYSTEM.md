# Black Market System

## The Underway

Every land has a different hidden entrance, but the routes join one shared market network called **The Underway**.

```text
Land-specific hidden route
→ generated safe Underway hub
→ rotating broker stock
→ exact return to source land
```

## Persistent state

Save v3 stores:

- Underworld Reputation.
- Discovered land routes.
- Forbidden Dungeon Keys.
- Active Anonymous Contracts.
- Remaining Clean Papers protection.
- Source land and return coordinates while inside the hub.

## Current offers

### Contraband Cache

Adds a Supply Crate and a resource bundle biased toward the source land.

### Lost Territory Map

Reveals the source land’s Lost Territory route on the world map.

### Clean Papers

Adds protection counters for future smuggler-route inspection systems.

### Forbidden Dungeon Key

Persists in the underworld inventory and opens a sealed-cache reward on the next dungeon entry.

### Anonymous Contract

Persists until the player reaches a dungeon’s final floor, then pays crystals and reputation.

## Stock rotation

Stock is deterministic from world day and reputation. This makes offers testable and consistent without requiring a live economy service in the prototype.
