# Ecosystem System

## Purpose

Wildlife should make each land feel inhabited rather than functioning only as loot containers.

## Current implementation

- Fourteen runtime animal kinds.
- Land-specific settlement animal pools.
- Passive collection timers.
- Combat hunting rewards.
- Resource-specific materials.
- Death/respawn cycle for renewable populations.
- Authored passive, predator, apex and legendary lists per land.
- Region generation multipliers for trees, water, crystal, iron, shrooms, enemies and ambushes.

Temporary sprites reuse cow/chicken animation families where a final species asset does not yet exist. This is intentional: gameplay data and behavior are being stabilized before asset production.

## Planned population model

The next ecosystem phase can add:

- Population budgets per region.
- Predator/prey pressure.
- Herd and pack movement.
- Day/night activity.
- Weather-driven migration.
- Nest/offspring protection.
- Taming, pets and mounts.
- Overhunting and recovery.
- Legendary creature event chains.

## Land examples

- The Witchlands: moth deer, black goats, corrupted predators.
- Green Land: red deer, boar, cattle and common birds.
- Rainforest: tapir, capybara, reptiles and jungle cats.
- Frostlands: reindeer, musk ox, ice wolves and bears.
- Sunscorched Desert: camels, gazelles, vipers and dune predators.
- Cinder Coast: shore goats, reef turtles, scavengers and volcanic creatures.
